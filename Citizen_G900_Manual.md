## 1. Features
- Minute repeater (electronic chime)
- Perpetual calendar (automatic leap‑year correction)
- Daily alarm
- Local time (second time zone)
- Local alarm
- Eco‑Drive solar power system

---

## 2. Names of Components
**Hands**
- Main watch: hour, minute, second
- Sub watch: hour, minute
- Local time / alarm hour hand
- Local time / alarm minute hand
- AM/PM hand
- Date & mode hand
- Month hand
- Year hand (years since last leap year)

**Controls**
- Crown
- Button (A): local time, month/year set & recall
- Button (B): minute repeater, date set, alarm ON/OFF
- Button (C): mode switch, reference position check

---

## 3. Before Using
### Checking the Reference Position
1. Crown in normal position
2. Hold Button (C) ~3 s → date/mode hand to `0‑SET`
3. Confirm:
   - Date & mode hand → `0‑SET`
   - Sub watch hour/minute → `12:00`
   - AM/PM hand → top of AM zone

---

## 4. Setting the Reference Position
### All‑Reset
1. Pull crown to Position 2
2. Press A + B + C for ≥2 s
3. Hands move and confirmation tone sounds

### Set Reference
1. Button (A) → sub hands to `12:00`, AM/PM at bottom PM
2. Button (B) → date/mode hand to `0‑SET`
3. Return crown to normal

---

## 5. Minute Repeater
- Press Button (B) ~1 s in time mode
- High pitch = hours
- High + low = 15‑minute blocks
- Low pitch = remaining minutes

---

## 6. Setting the Time and Date
### Main Watch Time
- Pull crown to Position 2 at 0 s
- Turn crown to set hours/minutes

### Sub Watch Time
- Button (A) to align with main time
- Check AM/PM

### Date
- Crown Position 1
- Button (A): month/year
- Button (B): date

---

## 7. Daily Alarm (H‑AL)
- Mode: Button (C)
- Set time: Button (A)
- ON/OFF: Button (B)
- Stop alarm: any button

---

## 8. Local Time (L‑TM)
- Mode: Button (C)
- Adjust in 30‑min steps with Button (A)

---

## 9. Local Alarm (L‑AL)
- Same operation as daily alarm, linked to local time

---

## 10. Seconds Correction (0‑SET)
- Mode: Button (C)
- Press Button (B) in sync with reference clock

---

## 11. Solar‑Powered Functions
- Low charge: second hand moves in 2‑s steps
- Overcharge prevention automatic

---

## 12. Charging Times (Approx.)
- Full charge reserve: ~9 months
- From stopped to running: ≥30 min

---

## 13. Handling Precautions
- Avoid charging above 60 °C
- Keep ≥50 cm from hot lamps

---

## 14. Secondary Battery
- Normally no replacement
- After replacement: All‑Reset mandatory

---

## 15. Troubleshooting
- Misaligned hands → All‑Reset
- 2‑s tick → recharge

---

## 16. Precautions
- Follow water‑resistance limits
- Do not operate crown/buttons when wet

---

## 17. Specifications
- Calibre: G900
- Accuracy: ±15 s/month
- Power reserve: ~9 months
- Battery: secondary rechargeable
